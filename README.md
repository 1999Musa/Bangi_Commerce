# Bangi E-Commerce

Bangi is a simple Laravel-based e-commerce application.

## Features

- Product listing
- Product details
- Add to cart
- Checkout process

## Tech Stack

- Laravel 11
- MySQL
- Bootstrap
- JavaScript
- REST API

## Installation

```bash
git clone https://github.com/1999Musa/Bangi_Commerce.git
cd Bangi_Commerce
composer install
cp .env.example .env
php artisan key:generate
php artisan migrate
php artisan serve
